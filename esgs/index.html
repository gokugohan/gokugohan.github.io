Objetivo

  • Criação de uma aplicação em android que assegura a gestão de despesas efetuadas por um grupo num determinado período de tempo, recolhendo as informações (despesas) em modo local.
  • Ao fechar as contas das despesas, esses dados serão enviados para servidor.

Descrição técnica

Tecnologias e ferramentas

  1. PC equipado com OS Microsoft windows 8.1 x64
  2. Java development kit 7 (jdk 7)
  3. Android Software Development Kit (Android sdk)
  4. Android Development Tools (eclipse adt) >>> Android Studio
  5. GenyMotion (Emulador virtual)
  6. Parse.com (Servidor remoto para receber dados enviados)
  7. Git
